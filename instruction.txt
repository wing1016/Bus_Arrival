WHAT: Objectives/Goals

Build a responsive web to show next bus nearby, for live tracking of bus arrivals, user choose a Hong Kong bus number and user GPS location. With these information input/select by user, the application fetch live bus information from https://data.gov.hk/. Web show the choosen bus number next 3 arrival times. User read this live bus arrival information at home before they go to the bus station everyday morning before they to work. 


WHAT: Users/Target 
Hong Kong citizen who go to work by bus.

 
WHY: ROI/Value
User can choose the best time to go to the bus station.


HOW: Technologies

Gateway: Node.js Express, authentication with JWT to build the middleware for authentication, authorization and routing, protect the intelligence backend
Intelligence: Python FastAPI, to connect powerful LLM to extract receipt details, generate insights
